# My-MC-Resource-Pack
This is a resource pack for Minecraft Bedrock Edition which gives elytra a new look and adds the grass texture on the side, making hills smoother.

# Installation (for Bedorck Edition)
[Download the MyPack.mcpack file](https://mobilefriendlyallay.github.io/My-MC-Resource-Pack/MyPack.mcpack), then, if the file has the Minecraft icon on it, simply click it. If it doesn't, right click > "open with app" and choose Minecraft. Then, in Minecraft, go to "settings" > "resourse packs" > "my packs" and click it, then you should see the "activate" button.
